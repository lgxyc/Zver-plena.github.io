完全背包问题

dp 模版 完全背包
Publish Date:   2021-11-01
Update Date:   2022-05-09

完全背包问题

题目

有 N 种物品和一个容量是 V 的背包，每种物品都有无限件可用。
第 i 种物品的体积是 vi，价值是 wi。
求解将哪些物品装入背包，可使这些物品的总体积不超过背包容量，且总价值最大。
输出最大价值。

模式

状态表示
f[i][j]表示前i个物品,背包容量为j的最大价值
状态转移

  1. 第i个物品选择0个
  2. 第i个物品选择1个
    ···
  3. 第i个物品选择k个

状态转移方程:f[i][j]=Max(f[i-1][j-k*v[i]]+k*w[i]) $k\in [0,1,2,3···k] $

状态属性
Max

时间复杂度
$O(nvk)$
暴力完全背包代码实现

#include <iostream>
#include <vector>
using namespace std;
const int N = 1010;
int f[N][N];
int main()
{
    int n, m;
    cin >> n >> m;
    vector<int> v(n + 1);
    vector<int> w(n + 1);
    for (int i = 1; i <= n; i++) {
        cin >> v[i] >> w[i];
    }
    //dp
    for (int i = 1; i <= n; i++) {
        for (int j = 0; j <= m; j++) {
            for (int k = 0; k * v[i] <= j; k++) {
            //不用判断j>=v[i],因为k*v[i]<=j已经判断过了
            //包括f[i][j]=f[i-1][j]包含在k=0的情况之中
                f[i][j] = max(f[i][j], f[i - 1][j - k * v[i]] + k * w[i]);
            }
        }
    }
    cout << f[n][m] << endl;

    return 0;
}

时间复杂度优化思路
f[i][j]=Max(f[i-1][j],f[i-1][j-v]+w,f[i-1][j-2v]+2w,f[i-1][j-3v]+3w····f[i-1][j-kv]+kw)
上述第二项开始对应下述第一项
f[i][j-v]=Max(f[i-1][j-v],f[i-1][j-2v]+w,f[i-1][j-3v]+2w····,f[i-1][j-kv]+(k-1)w)
很显然的等价替换发现
f[i][j]=Max(f[i-1][j],f[i][j-v]+w)
所以有了这一层关系,那么我们就不需要去循环k次
于是就可以优化了时间复杂度

时间复杂度
$O(nv)$

时间优化后代码如下

  //dp
for (int i = 1; i <= n; i++) {
      for (int j = 0; j <= m; j++) {
          f[i][j] = f[i - 1][j];
          if (j >= v[i]) {
              f[i][j] = max(f[i][j], f[i][j - v[i]] + w[i]);
          }
      }
  }

空间复杂度优化思路
同01背包优化的思路
我们同样本层的信息f[i][j]由本层的信息f[i][j]和本层的f[i][j-v[i]转移过来
参考01背包的
f[i][j]=max(f[i][j],f[i-1][j-v[i]]+w[i]
完全背包
f[i][j]=max(f[i][j],f[i][j-v[i]]+w[i])
可以很明显的发现只有后面有一处是不同
类似01背包的优化
与01背包的不同的是

  1. 01背包优化的j从m枚举到v[i]
  2. 完全背包优化的j从v[i]枚举到m;

因为01背包的状态是本层i是由上一层i-1转移过来的,而完全背包是本层i是由本层i转移过来.所以j从小到大枚举,确保j-v[i]是被本层修改后的信息

空间优化后代码如下

//dp
for (int i = 1; i <= n; i++) {
//与01背包不同的是从v[i]枚举到m
        for (int j = v[i]; j <= m; j++) {
            f[j] = max(f[j], f[j - v[i]] + w[i]);
        }
    }

经过时间和空间优化后的完全背包

代码如下

#include <iostream>
#include <vector>
using namespace std;
const int N = 1010;
int f[N];
int main()
{
    int n, m;
    cin >> n >> m;
    vector<int> v(n + 1);
    vector<int> w(n + 1);
    for (int i = 1; i <= n; i++) {
        cin >> v[i] >> w[i];
    }
    for (int i = 1; i <= n; i++) 
        for (int j = v[i]; j <= m; j++) {
            f[j] = max(f[j], f[j - v[i]] + w[i]);
        }
    cout << f[m] << endl;
    return 0;
}
